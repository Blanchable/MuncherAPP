# Assets Directory

This directory contains all the static assets for the Muncher app.

## Required Files

### Icons
- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (32x32px)

### Splash Screen
- `splash.png` - Splash screen image (1242x2436px)

## Image Specifications

### App Icon
- **Size**: 1024x1024 pixels
- **Format**: PNG
- **Background**: Transparent or solid color
- **Design**: Should be recognizable at small sizes

### Adaptive Icon (Android)
- **Size**: 1024x1024 pixels
- **Format**: PNG
- **Safe Zone**: Keep important elements within the center 66% of the image
- **Background**: Should work with different shapes

### Splash Screen
- **Size**: 1242x2436 pixels (iPhone X resolution)
- **Format**: PNG
- **Content**: App logo and name
- **Background**: Should match app theme

## Design Guidelines

1. **Consistency**: All assets should follow the same design language
2. **Accessibility**: Ensure sufficient contrast for visibility
3. **Scalability**: Icons should be recognizable at various sizes
4. **Branding**: Maintain consistent brand colors and typography

## File Naming

Use lowercase letters and hyphens for file names:
- `app-icon.png`
- `splash-screen.png`
- `favicon.png`

## Optimization

Before adding files to this directory:
1. Optimize images for web/mobile
2. Use appropriate compression
3. Test on different devices
4. Ensure fast loading times
